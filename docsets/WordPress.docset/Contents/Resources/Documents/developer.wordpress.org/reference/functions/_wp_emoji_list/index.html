Description #Description

These arrays are automatically built from the regex in twemoji.js – if they need to be updated, you should update the regex there, then run the npm run grunt precommit:emoji job.

Top ↑

Parameters #Parameters

$type

(string) (Optional) Which array type to return. Accepts 'partials' or 'entities', default 'entities'.

Default value: 'entities'

Top ↑

Return #Return

(array) An array to match all emoji that WordPress recognises.

Описание

Возвращает одиночную запись, основываясь на ID записи.

Использование

<?php wp_get_single_post( $postid, $mode ) ?>

Параметры

$postid
(integer) (необязательный) ID записи.
Default: 0
$mode
(string) (необязательный) Как возвращать результат. Ожидает константу: OBJECT, ARRAY_N или ARRAY_A.
Default: OBJECT

Возвращаемые значения

(object|array) 
Объект записи или массив, включающий содержание записи и информацию с двумя добавочными полями (или ключами): 'post_category' и 'tags_input'.

Примеры

Примечания

  • Uses: get_post()
  • Uses: wp_get_post_categories()
  • Uses: wp_get_post_tags()

Change Log

  • Deprecated: 3.5.0
  • Since: 1.0.0

Source File

wp_get_single_post() is located in wp-includes/post.php.

Añadiendo Menús de Administración

Introducción

Normalmente, los autores de plugins y themes necesitan dar acceso a las paginas de configuraciones (opciones) para que los usuarios puedan personalizar el modo en que el plugin o theme es utilizado. La mejor manera para presentarle al usuario con tal pantalla es creando un menú de administración que permita al usuario tener acceso a la pagina de opciones. En este artículo se explica cómo los autores de plugins pueden añadir menús y pantallas de administración. Nota: la siguiente información asume un conocimiento de lo básico de Escribiendo un Plugin y usando el Plugin API de Acciones y Filtros.

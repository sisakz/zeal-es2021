Описание

Этот тег шаблона показывает ссылки на метку или метки записи. Если меток нет, вместо них будет показана связанная рубрика. Эта функция должна быть внутри Цикла.

Использование

 <?php the_tags( $before, $separator, $after ); ?> 

Параметры

$before 
(string) Текст для показа перед метками. По умолчанию Метки:
$separator 
(string) Текст или символ для показа между ссылками на метки. По умолчанию запятая (,).
$after 
(string) Текст для показа после списка меток. По умолчанию ничего не выводится.

Примеры

По умолчанию

По умолчанию используется список меток через запятую (,) и со словом Метки: в начале. 

<p><?php the_tags(); ?></p>
Метки: WordPress, Компьютеры, Блоггинг

Разделенные запятой

Выводится список меток с переносом строки в конце. 

 <?php the_tags('Метки:', ', ', '<br />'); ?>

Разделённые стрелкой

Выводит список меток, разделённых стрелкой (>). 

<?php the_tags('Метки: ',' > '); ?>
Метки: WordPress > Компьютеры > Блоггинг

Разделённые круглым маркером

Выводит список меток, разделённых круглым маркером (•), с переносом строки в конце. 

<?php the_tags('Метки: ',' • ','<br />'); ?>
Метки: WordPress • Компьютеры • Блоггинг

Списком

Выводит метки в виде неупорядоченного списка. 

<?php the_tags('<ul><li>','</li><li>','</li></ul>'); ?>

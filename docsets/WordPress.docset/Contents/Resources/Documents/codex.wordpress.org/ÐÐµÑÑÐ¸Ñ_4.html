Версия 4.2

Languages: English • Русский • (Add your language)

В апреле 2015, WordPress версии 4.2, названный jazz musician xxxxx, будет доступен для общего выпуска. для большей информации о новых выпусках и исправлениях ошибок читайте WordPress Blog, и смотрите Список изменений в версии 4.2.

Для версии 4.2, версия базы данных (db_version в wp_options) поменялась на xxxxx, и последняя сборка в Trac была xxxxx.

Общие изменения

  • По мелочи
    • Press This был полностью переработан, чтобы сделать публикацию текстов с других сайтов проще, чем когда либо
    • Теперь вы можете устанавливать и переключать темы прямо в Настройщике
    • Больше интуитивных обновлений и установок на экране плагинов
    • Добавлена поддержка Emoji для записей
  • Для разработчиков
    • Общие термины нескольких таксономий теперь делятся на разные параметры, при обновлении одного из них
    • Сложные запросы - поддержка конкретных meta_query классов, которые могут быть использованы с orderby, добавлено в WP_Query, WP_Comment_Query, и WP_User_Query
    • Темы теперь могут использовать HTML5 Markup для Виджетов, using используя add_theme_support( 'html5', array( 'widgets' ) )
